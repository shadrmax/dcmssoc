DCMS-Social 1.9.13
------------------------------------------------------
Над релизом работали: biket, Hryhorii_Ivlev


Скрипт движка для вашего WEB/WAP – сайта

Требования: PHP 5.6 и выше (рекомендуется PHP 7.x), MySQL, поддержка mod_rewrite,  платный хостинг.

Если у вас получилось установить и полноценно использовать движок на бесплатном хостинге, просим вас сообщить об этом на форуме официального сайта проекта: https://dcms-social.ru/forum/

Рекомендуемые библиотеки (при отсутствии этих библиотек могут отсутствовать некоторые возможности):


1)	Iconv
2)	Ffmpeg
3)	GD
4)	mcrypt

Модули:

1)	Чат (Умник+1000 вопросов, Шутник+1000 шуток).
2)	Форум (2 уровня, Прикрепление файлов, Поиск, Закладки).
3)	Центр загрузок (Неограниченное кол-во вложенных папок, выгрузка, импорт, скриншоты, комментарии, счетчик скачиваний с прямым путем к файлу.)
4)	Файловый обменник (корректная поддержка русских названий файлов и папок, неограниченное количество вложенных папок, скриншоты, информация о файлах, индивидуальные настройки для выгрузки для каждой папки).
5)	Библиотека
6)	Новости с RSS
7)	Гостевая
8)	Голосования

Основные папки и файлы движка:

• Файлы, прикрепленные в форуме: sys/forum/files/(*.frf)
• Файлы обменника: sys/obmen/files/(*.dat)
• Темы оформления: style/themes/(папка с темой)
• Правила сайта: sys/add/rules.txt
• Архив темы по умолчанию: sys/add/them.zip (используется для подстановки недостающих файлов темы при установке темы через админку)


Установка:

1)  Создаем базу данных MySQL (именно базу, а не таблицу).
2)  Заливаем все файлы к себе на хост в КОРНЕВУЮ директорию или в папку поддомена. (Движок не будет работать в подпапке).
3)  Переходим по адресу https://[ваш сайт]/install/
4)  Следуем всем пунктам установки.
5)  Если у Вас возникли трудности с дальнейшей установкой или у Вас возникли предложения по улучшению движка, то просим посетить наш форум http://dcms-social.ru/forum/

Дополнительные модули можно будет скачать и установить отдельно вручную.
